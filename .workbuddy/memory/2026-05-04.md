# 2026-05-04

## mem0-demo 多 Agent 架构优化

### 文件重命名
- `agent.py` → `knowledge_agent.py`
- `orchestrator.py` → `router_agent.py`

### 串行组合路由
- `_route()` 从返回单字符串改为返回 `list[str]`
- 关键词规则改为同时检测（不短路），同时命中两个时都加入列表
- LLM 路由 prompt 支持返回多目标（如 `knowledge,merchant`）
- `chat()` 串行调用多个 Agent，`_merge_answers()` 去重合并结果
- `stream_chat()` 串行流式，多 Agent 间用 `\n\n` 分隔
