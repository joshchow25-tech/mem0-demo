# 产品手册

本手册介绍我们电商平台的核心功能和使用指南。

## 注册与登录

### 注册流程

1. 访问官网点击「免费注册」
2. 填写手机号并获取验证码
3. 设置登录密码（8位以上，含字母和数字）
4. 完善个人信息即可完成注册

### 登录方式

支持以下登录方式：
- 手机号 + 密码
- 手机号 + 短信验证码
- 微信快捷登录

## 下单流程

### 选购商品

在商品详情页选择规格（颜色、尺码等），点击「加入购物车」或「立即购买」。

### 提交订单

1. 确认收货地址
2. 选择配送方式
3. 选择优惠券或使用积分抵扣
4. 选择支付方式并完成支付

### 订单状态

| 状态 | 说明 |
|------|------|
| 待付款 | 已下单，等待付款（24小时内） |
| 待发货 | 付款成功，商家备货中 |
| 已发货 | 商品已出库，运输中 |
| 已完成 | 确认收货，交易完成 |
| 已取消 | 订单已取消 |

## 退换货政策

### 无理由退货

收货后 **7天内**，商品未使用、包装完好，可申请无理由退货。

以下商品不支持无理由退货：
- 食品、生鲜类
- 定制类商品
- 数字内容（虚拟商品）

### 质量问题

收货后 **30天内**，因质量问题可申请免费换货或退款，运费由商家承担。

## 会员体系

### 等级说明

| 等级 | 累计消费 | 折扣 | 专属权益 |
|------|----------|------|----------|
| 普通会员 | 0元起 | 无 | 积分返现 |
| 银卡会员 | ≥1000元 | 9.8折 | 生日礼包 |
| 金卡会员 | ≥5000元 | 9.5折 | 专属客服 |
| 钻石会员 | ≥20000元 | 9折 | 免费包邮 |

### 积分规则

- 每消费 1元 = 1积分
- 100积分 = 1元优惠券
- 积分有效期为获取后 2年
